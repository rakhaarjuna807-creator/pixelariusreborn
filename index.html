How to Join

Server Address

play.pixelarius.web.id

Java Edition

  1. Open Minecraft and select "Multiplayer"
  2. Click "Add Server"
  3. Enter any name and our server address
  4. Click "Done" and join the server!

Bedrock Edition

  1. Go to "Play" and select "Servers"
  2. Scroll down and click "Add Server"
  3. Enter our server address and port 19132
  4. Save and connect to the server!
